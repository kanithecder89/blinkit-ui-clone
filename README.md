# Blinkit UI Clone 

A frontend clone of the Blinkit interface focusing on layout accuracy and responsive design.

## Features
- Pixel-perfect UI replication
- Responsive across devices
- Structured layout and styling

## Tech Stack
HTML5, CSS3, JavaScript

##  Purpose
To practice real-world UI cloning and responsive frontend development.
